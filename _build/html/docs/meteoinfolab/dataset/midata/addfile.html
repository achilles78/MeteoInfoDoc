addfile¶

mipylib.dataset.midata.addfile(fname, access='r', dtype='netcdf')¶

Opens a data file that is written in a supported file format.

Parameters:
  • fname – (string) The full or relative path of the data file to load.
  • access – (string) The access right setting to the data file. Default is r.
  • dtype – (string) The data type of the data file. Default is netcdf.
Returns:

(DimDataFile) Opened file object.

Examples
