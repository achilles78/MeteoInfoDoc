Introduction¶

MeteoInfoLab is a free software product developed using Java and Jython based on MeteoInfo Java library (Unidata netCDF Java library is used). The purpose is to provide an optional scientific computation and visualization tool similar with MatLab and/or NCL. You can download the lastest MeteoInfo Java software from the website: http://www.meteothinker.com/. Then unzip the file and run the MeteoInfoLab.exe file in MeteoInfo folder. For Mac or Linux/Unix user, MeteoInfoLab can be run from a terminal using milab.sh or using the command: java -jar MeteoInfoLab.jar. Java 7 or better is needed.
